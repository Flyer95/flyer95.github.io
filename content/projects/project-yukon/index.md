### *Solo project created in 2020. Still in development.*

Project Yukon is a third-person adventure where you work together with a fox to solve puzzles and discover the history of an abandoned civilization.

I've been working on this project in my free time since 2020. Through the project, I've improved my skills in using the Unreal Engine and Blueprints in terms of AI behavior trees, procedural animations, and interfaces for use in player-companion interactions.

In addition, I've learned a lot game design principles and practices, such as creating level blockouts, tying storytelling to gameplay, and maintaining a game design document.
